Fichiers Javascript de TD

Chaque programme demandé en TD est dans son propre fichier de type Javascript (extension .js).
Ouvrez un fichier TD dans un éditeur de texte pour lire ses commentaires, le comprendre et le modifier.
Vous y trouverez quelque chose comme ça :
//cette ligne est commentée grâce aux '//' est n'est donc pas exécutée

//ci-dessous vos registres
rm.etat = {
//choix des valeurs
//changez les et observez l'état final résultant
R1: 777,
R2: 888
};

rm.programme_NOM-PROGRAMME = function() {
//Programme : ici doivent être vos instructions
}

Instructions de TD écrites en Javascript

  • Remise à 0 (reset) : rm.remise_a_zero('R'x);

    Exemple : rm.remise_a_zero('R1');

  • Incrément (++) : rm.increment('R'x);

    Exemple : rm.increment('R1');

  • Tant que != : rm.tant_que_different('R'x, 'R'y,
    function() { // faire
    //Programme tant que
    }
    );

    Exemple : rm.tant_que_different('R1', 'R2',
    function() { // faire
    rm.increment('R2'); //R2++
    }
    );
