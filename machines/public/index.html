d) Mettre en pause un programme : breakpointer

Vous pouvez à toute ligne mettre en pause le programme pour vérifier les valeurs connues de l'ordinateur à cet instant précis. Dans le code Javascript, avant la ligne où vous souhaitez que l'ordinateur se mette en pause, placez un breakpoint en écrivant :
debugger;
Si les outils développeur du navigateur sont ouverts (sinon ouvrez-les et rafraîchissez la page), vous verrez l'interface afficher la ligne et les possibilités de voir les variables ("registres") utilisées comme continuer l'exécution du code jusqu'au prochain breakpoint atteint.

2. TD1 : Machine à registres

Équivalent des instructions de TD en Javascript

  • Remise à 0 (reset) : rm.remise_a_zero('R'x); //où x est le numéro du registre à mettre à 0

    Exemple : rm.remise_a_zero('R1');

  • Incrément (++) : rm.increment('R'x); //où x est le numéro du registre à incrémenter

    Exemple : rm.increment('R1');

  • Tant que Rx!=Ry :
    //où x et y sont les numéros des registres à comparer rm.tant_que_different('R'x, 'R'y,
      function() { // faire
        //Programme tant que
      }
    );

    Exemple : rm.tant_que_different('R1', 'R2',
      function() { // faire
        rm.increment('R2'); //R2++
      }
    );

3. TD2 : Machine de Peano

Les expressions données dans le TD peuvent se coder directement en Javascript avec de minimes modifications :

  • ++expr : si expr est une constante: expr+1 sinon: ++expr
  • --expr : si expr est une constante: expr-1 sinon: --expr
  • Définir une fonction : function NOM_DE_MA_FONCTION(ARG0, ARG1, ARG2, ...){}
  • Dans une fonction, retourner une valeur : return expr;
Ainsi nous pourrions avoir :
expr = 0;
1+expr; //expr = 1
--expr; //expr = 0
function NOM_DE_MA_FONCTION(ARG0, ARG1, ARG2, ...){
 return 0; }
